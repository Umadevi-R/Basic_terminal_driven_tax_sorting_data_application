PROJECT TITLE:
        Tax-data-sorting application
PURPOSE OF PROJECT:
     1.This project is a  basic terminal driven application that can manipulate the dataset to get a list of counties in Missouri in ascending order of their tax rate.
     2.This project has 5 different sorting techniques namely(BUBBLE SORT,SELECTION SORT,INSERTION SORT,MERGE SORT,QUICK SORT)
     3.The project also displays the time taken to sort the data set and thus helps us to analyze the effiency of the different orting techniques.
